linux bash 命令查遗补漏

  • ver 显示dos系统版本号
  • md(mkdir) 创建文件夹
  • ren(rename) 修改文件夹
    ren aaa zhai
  • cat 将文件内容输入到终端上，只能在bash里用
  • diff 比较两文件不同
  • kill 9 杀进程
  • dir 展示目录详细信息
  • grep 用于查找文件中符合条件的字符串
    grep hexo --color=auto *md 查找后缀为md格式的文件中包含hexo字符串的文件，并打印出来，hexo标红
  • find 查找
  • ps 列出进程信息 -l -x -a

  • tar 打包文件

  • time 测算一个命令运行时间
